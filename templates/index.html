About Visual Price Checker

Visual Price Checker is a smart tool designed to help users find the best deals online. By leveraging advanced image recognition and cloud vision technology, it allows you to compare prices effortlessly and save time.

Our mission is to make shopping smarter, easier, and more transparent. We focus on accuracy, efficiency, and simplicity so you can enjoy a hassle-free price comparison experience.

This project was built as a mini project to demonstrate the power of modern web technologies like Flask, Dropzone.js, and Cloud Vision API. We aim to continually improve and innovate for a seamless user experience.
